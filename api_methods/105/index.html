Export an API :: /apis/{apiName}?format=bundle

API resource: APIs

Verb: GET

Doc: Outputs an API revision as a ZIP bundle of code and con fig files. This enables local development, attachment of policies and scripts. To export the ZIP file to a local file add '> filename.zip' to the curl command above to create the named ZIP in the current working directory.

Description: Exports an API from Apigee to the local machine a ZIP bundle of config and code files.

cURL: 
curl -u myname:mypass https://api.enterprise.apigee.com/v1/o/{organizationName}/apis/{apiName}/revisions/{revisionNumber}?format=bundle
