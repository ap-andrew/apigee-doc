Get Developer of an App :: /developers?app={appName}

API resource: Developers

Verb: GET

Doc: This call enables you to retrieve a Developer's profile based on the name of an App. The profile retrieved is for the Developer who registered the App in your Organization.

Description: Gets the Developer profile by App name

cURL: 
curl -u myname:mypass https://api.enterprise.apigee.com/v2/organizations/{organizationName}/developers?app={appName}

Request Payload Elements:
Name Description Default Required?

Response Payload Elements:
Name Description
apps: The name of the App Detail
email: The email address of the Developer who registered the App Detail
firstName: The first name of the Developer who registered the App Detail
last_name: The last name of the Developer who registered the App Detail
userName: The username of the Developer who registered the App (unused by Apigee) Detail
organizationName: The Organization which the Developer who registered the App belongs Detail
status: The status of the Developer who registered the App (active, inactive, or login_lock Detail
attributes: Arbitrary name/value attributes that are part of the Developer profile Detail
createdAt: The Unix time at which the Developer profile was created Detail
createdBy: The Apigee username of the user who created the Developer profile Detail
lastModifiedAt: The Unix time at which the Developer profile was last modified Detail
lastModifiedAt: The Apigee username of the user who most recently modified the Developer profile Detail
Errors
Auth: Basic Auth
