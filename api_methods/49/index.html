API resource: Company App Keys

Verb: POST

Doc: Update existing App Key with New ApiProduct(s). New ApiProduct(s) will merge existing ApiProduct(s).

Description: Update existing App Key with New ApiProduct(s).

curl -u myname:mypass https://api.enterprise.apigee.com/v1/organization/sears/companies/devcompany/apps/myapp/keys/1234567890
