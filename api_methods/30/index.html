Update a Developer App :: /developers/{developerName}/apps/{appName}

API resource: Developer Apps

Verb: PUT

Doc: When a developer wants to register an app to consume your API product, you create a developer app in your organization within Apigee Enterprise. The developer creates and registers an app using an API product of an organization and gets the consumer key in the response. Use the POST method to create developer app.

Description: Updates the app to get a new set of consumer credentials.

cURL: 
curl -u {myname:mypass} "https://api.enterprise.apigee.com/v1/o/{myorg}/developers/{developerName}/apps/{appName}" -X POST -d'
