Update Access Token :: /oauth2/accesstokens/{accessToken}

API resource: Oauth 2.0 Access Tokens

Verb: PUT

Doc: Updates the access token attributes. 1. Can update attributes of Authorization Code and AccessToken 2. Only attributes given in payload will be updated and the remaining existing attributes will remain intact.

Description: Updates the access token attributes.

curl -u myname:mypass https://api.enterprise.apigee.com/v1/o/{myorg}/oauth2/accesstokens/{accessToken}
