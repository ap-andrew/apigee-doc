List Developers :: /developers

API resource: Developers

Verb: GET

Doc: Returns an array of email addresses, each of which is the unique key for a developer in the Organization the call is made against. This call does not list any Company Developers who may be part of an Organization.

Description: Lists all developers by email address (the unique key for developers in Apigee)

cURL: 
curl -u myname:mypass https://api.enterprise.apigee.com/v2/organizations/{organizationName}/developers
