API Products

/apiproducts
A bundle of API resources (URIs) plus rate limit settings and relevant metadata
Apigee Enterprise provides API products to enable you to bundle and distribute your APIs to varied developers on the fly, with having to change any code. As part of the app provisioning flow in Apigee Enterprise, developers are presented with one or more API products to select. The developer app is provisioned with a key and secret that enable the app to access the resources defined wihin the API product. The API supports multiple API products per app key. API products provide the basis for access control--at least one API product is required to enable developers to invoke to your APIs.
