Oauth 1.0 Access Tokens for an App
Apigee Enterprise validates the API key, retrieves the service profile associated with the API key, and checks to see that the quota is not exceeded. When you apply OAuth to the API, Enterprise checks the request for an access token. If an access token is present, and the API is within the scope of the access token, you are then allowed to access the API.
